🎓 College FAQ Chatbot

Description

This is a simple FAQ Chatbot built using Python and Streamlit. It uses NLP techniques like TF-IDF Vectorization and Cosine Similarity to match user questions with the most relevant answers.

Features

* FAQ-based question answering system
* NLP similarity matching
* Streamlit chat interface
* Fast and simple responses

Technologies Used

* Python
* Streamlit
* Scikit-learn

How to Run

Install dependencies:

pip install -r requirements.txt

Run the app:

streamlit run app.py
